# Claude Code エディタ設定

このプロジェクトではClaude CodeがVS Codeエディタ内で常時利用可能になるよう設定されています。

## 使用方法

- `Cmd+Esc` (Mac) または `Ctrl+Esc` (Windows/Linux) でチャットパネルを開く
- `Cmd+Option+K` でファイルをClaude Codeに送信
- コードを選択すると自動的にClaude Codeに共有される

## 設定ファイル

- `.claude/settings.json` - Claude Code固有の設定
- `.vscode/settings.json` - VS Code統合設定
- `CLAUDE.md` - プロジェクト文脈とメモリ
